# Bookstore Management System

A professional REST API for managing an online bookstore, built with Spring Boot.

---

##  Features

- User Registration & Login
- JWT Authentication
- Role-Based Authorization (Admin & Customer)
- Book Management (CRUD)
- Order Management
- MySQL Database Integration
- Pagination & Search
- Global Exception Handling
- Swagger API Documentation
- Unit Testing with JUnit

---

##  Tech Stack

- Java 21
- Spring Boot 3
- Spring Security
- Spring Data JPA
- MySQL
- JWT
- Maven
- Lombok
- Swagger (OpenAPI)
- JUnit
- Postman
